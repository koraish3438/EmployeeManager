# EmployeeManager

A professional **Employee Management Android App** built with **Kotlin**, **Room (with KSP)**, **ViewBinding**, **RecyclerView**, and **Coil** for image loading. This app allows adding, editing, viewing, and deleting employees with a clean and responsive UI.

---

## Project Overview

EmployeeManager is a modern Android application designed to simplify employee management for organizations or small businesses. Users can add new employees, edit existing employee information, and view detailed profiles.  

- **Purpose:** To make employee data management simple and efficient.  
- **Target Audience:** Small businesses, HR teams, individual managers.  
- **Impact:** Digitizes employee management, organizes records, and improves efficiency.

---

## Working Flow

1. **Employee List:** Displays all employees in a clean list.  
2. **Add Employee:** Add new employee information.  
3. **Edit Employee:** Edit existing employee details.  
4. **Employee Details:** View full employee profile with statistics.  
5. **Delete Employee:** Remove unwanted employee records.  

All actions are **stored locally** using **Room Database**, with a responsive RecyclerView interface.

---

## Features

- Add, edit, delete employees  
- View employee details with profile images  
- Local persistence using Room Database  
- Clean and responsive UI with ViewBinding  
- Images loaded using Coil library
